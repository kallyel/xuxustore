Fazer Pedido

Atores

Cliente

Pré-condição:

O usuário necessita ter feito o login no sistema.

Fluxo de Eventos principal

  1. O cliente fornece seu nome e endereço, e se possui cartão da loja
  2. O sistema devolve as descrições e o preço de cada produto.
  3. O cliente escolhe a forma de pagamento.
    • O cliente pode escolher o pagamento via boleto.
    • O cliente pode escolher o pagamento via PayPal.
    • O cliente pode escolher o pagamento via cartão de crédito.

Pós-condição:

O pedido irá ser adicionado ao sistema.
