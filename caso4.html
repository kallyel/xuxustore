Cancelar Pedido

Atores

Cliente

Funcionário

Pré-condição:

O usuário necessita ter feito o pedido.

Fluxo de Eventos principal

  1. O caso de uso começa quando o cliente seleciona "Cancelar Pedido".
  2. O cliente usa Procurar Pedido
  3. O Sistema mostra os dados da situação do pedido e o caso de uso termina.

Pós-condição:

A situação do pedido é marcada como cancelada. O registro do pedido permanece para o histórico de pedidos.
