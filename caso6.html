Fornecer Produto

Atores

Fornecedor

Pré-condição:

Os fornecedores devem ser aptos para fornecer.

Fluxo de Eventos principal

  1. O Fornecedor precisa entregar uma lista com seus produtos, para serem analisados e escolhidos qual fornecedor é mais barato.
  2. O sistema notifica o fornecedor que foi escolhido para fornecer.
  3. Ele mandará os produtos para a loja.

Pós-condição:

Os produtos chegarem na loja.
