Procurar Pedido

Atores

Cliente

Funcionário

Pré-condição:

O usuário necessita ter feito pedidos

O usuário necessita estar logado

Fluxo de Eventos principal

  1. O caso começa ao clicar em "Buscar Pedidos"
  2. Uma página é aberta demonstrando todos os pedidos realizados pelo cliente.
  3. O usuário clica no pedido que deseja

Pós-condição:

O usuário será redirecionado para a página mostrando o Status do pedido.
